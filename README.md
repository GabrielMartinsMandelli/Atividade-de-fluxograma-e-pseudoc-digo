# Atividade de Lógica 🚀

Nessa atividade, nessa atividade eu montei um fluxograma e pseudocódigo. Foi passado 5 situações comuns e transformei cada uma em fluxogramas e pseudocódigos para entender como o "passo a passo" funciona.

## 📝 O que eu analisei:

* **Fazer um Bolo de Cenoura 🍰** (Seguir a receita na ordem certa)
* **Ir para o Trabalho 🚶‍♂️** (Decidir o melhor caminho)
* **Trocar o Pneu do Carro 🚗** (Segurança e esforço manual)
* **Trocar uma Lâmpada 💡** (Resolver um problema simples)
* **Pegar um Ônibus 🚌** (Lidar com horários e passagens)

---

## O que eu aprendi de verdade

Muita gente acha que programação é só escrever código difícil, mas nessa atividade eu vi que é muito mais sobre organizar . Olha só o que eu aprendi:

###  Desenhar ajuda a entender (Fluxograma)
Usar as formas (quadradinho para ação, losango para pergunta) ajuda muito a ver onde o processo pode dar errado.

### As regras do "Se" e do "Enquanto"
No pseudocódigo, eu entendi como as decisões funcionam.

### Atenção aos detalhes
Percebi que se a gente esquece um detalhe bobo o algoritmo todo falha. Isso me ajudou a ter mais atenção com a sequência das coisas.

##  Conclusão
Essa atividade foi massa porque me mostrou que eu já uso lógica o tempo todo na minha vida, só não sabia os nomes técnicos. Agora ficou bem mais fácil entender como um programa de computador pensa.
